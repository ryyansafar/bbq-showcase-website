# 🥩 THE SMOKEHOUSE: An Immersive BBQ Showcase

An immersive, cinematic web experience designed for a premium, high-end smokehouse concept. This project focuses on **motion-driven storytelling**, **interactive micro-interactions**, and **premium performance**.

## 🚀 Live Demo
Visit the live showcase: [**smoke.ryyansafar.site**](https://smoke.ryyansafar.site)

---

## ✨ Key Features

- **Cinematic UI/UX**: High-fidelity hero sections with video backgrounds, layered depth effects, and glassmorphism.
- **Motion-Driven Storytelling**: Smooth scroll-based animations and transitions using Framer Motion.
- **Interactive Micro-interactions**: Magnetic buttons, hover physics, and subtle UI state transitions.
- **Crafted Typography**: A sophisticated hierarchy using Oswald for displays and Plus Jakarta Sans for body text.
- **Performance Optimized**: Built with Next.js App Router for server-side rendering and optimized image delivery.

## 🏗️ Getting Started

### Prerequisites
- Node.js 20.x or higher
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ryyansafar/bbq-showcase-website.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

The project was built with a "Quality over Quantity" mindset. Every interaction, from the loader sequence to the parallax backgrounds, is designed to feel **intentional, weighted, and premium**. 

The goal was to create a digital extension of a physical smokehouse—where fire, wood, and patience are the core ingredients. We translated these through:
- **Glowing Ambient Lighting**: Representing the warmth of live fire.
- **Inertia Scrolling**: Mimicking the slow, deliberate nature of the smoking process.
- **Minimalist Layouts**: Allowing the high-quality food photography to remain the focal point.
